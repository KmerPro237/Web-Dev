Strategiespiele


Strategiespiele sind Brettspiele, bei denen der Zufall keine Rolle für den Ausgang des Spiels spielt.
Die meisten Strategiespiele werden von zwei Spielern gespielt. Dabei werden ein Brett und Figuren (vor "Figuren" kein Zeilenumbruch: durch Ändern der Fenstergröße ausprobieren) verwendet.

Schach

Schach wird auf einem quadratischen Brett mit 64 Feldern gespielt.

Go

Go wird auf einem rechteckigen Gitter mit 361 Schnittpunkten gespielt.
